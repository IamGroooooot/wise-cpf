# 슬기로운 탄단지 (wise-cpf)
슬기로운 탄단지, Wise CPF(Carbohydrate Protein Fat의 약자), 프로젝트 - 사회적 기업 만들어보기

## 실행하는 법
 - 먼저 CLI에 Gatsby 깔기 
```sh
npm install --global gatsby-cli
```

 - CLI에서 실행
```sh
cd gatsby-directory
npm install
gatsby develop
```

## 빌드하는 법
 - config.js에서 경로 설정 후
 - 경로 포함해서 빌드
```
gatsby build --prefix-paths
```
